# Beep Sound Placeholder

This directory should contain a short beep sound file named `beep.mp3` for the capture feedback.

The sound should be:
- Short (100-200ms)
- Pleasant beep or click sound
- Used for audio feedback when a bottle is auto-captured

You can generate a simple beep using:
- Online tone generators
- Audio editing software
- Or use a system sound

Recommended: A short "ding" or "click" sound around 100-150ms duration.
